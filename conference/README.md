# Conference Materials

This directory contains materials for conference presentations, papers, and demonstrations about the ESP32 LoRa Sniffer project.

## Contents

- **SESSION_KEY_HARVESTING_GUIDE.md** - Technical deep dive into session key cryptanalysis
- **TEXT_MESSAGE_COMPLETE_GUIDE.md** - Text message decryption methodology
- **README_SECURITY.md** - Security research positioning and pitch
- **HARDWARE_STRESS_TESTING_GUIDE.md** - Attack surface analysis methodology
- **tdeck_port_guide.md** - Hardware variant porting guide

## Usage

These documents are intended for:
- Conference presentations (DEF CON, Black Hat, etc.)
- Academic papers on LoRa/Meshtastic security
- Security research documentation
- Technical deep-dives for advanced audiences
